# 🎵 Music Downloader Bot

Telegram-бот для скачивания музыки с YouTube.

## 🚀 Быстрый старт на Render

1. Форкни этот репозиторий на GitHub
2. Зарегистрируйся на [render.com](https://render.com) через GitHub
3. Создай новый Web Service → Connect GitHub → выбери этот репозиторий
4. В настройках добавь Environment Variable: `BOT_TOKEN=твой_токен`
5. Нажми Deploy!

## 📱 Как пользоваться

1. Отправь боту название трека
2. Выбери из найденных вариантов
3. Получи MP3 файл!

## ⚠️ Важно

- Бот работает на бесплатном тарифе Render — может "засыпать" после 15 минут бездействия
- Первый запрос после сна может занять 1-2 минуты
- Лимит Telegram: файлы до 50 MB
